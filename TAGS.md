# 标签规范

## 使用规则
- 每次打标签时，优先从已有标签中选择。
- 如需新增标签，必须先在此文件中登记含义，再使用。

## 现有标签列表

| 标签名 | 含义 | 示例笔记 |
|--------|------|----------|
| `代理` | 各类代理技术（正向/反向/透明） | 透明代理、代理链 |
| `隧道` | DNS/HTTP/ICMP等隧道技术 | DNS隧道、ICMP隧道 |
| `C2` | 命令与控制相关 | C2隐匿、域前置 |
| `抓包` | 流量捕获与分析 | tcpdump、Wireshark |
| `绕过` | 安全策略绕过技巧 | 防火墙绕过、AV绕过 |
| `检测` | 威胁检测与流量分析 | DNS隧道检测、C2特征识别 |
| `iptables` | Linux防火墙配置 | 透明代理、端口转发 |
| `DNS` | 域名系统相关 | DNS隧道、DNSSEC |
| `TPROXY` | 透明代理技术 | TPROXY实战 |
| `Docker` | 容器化环境搭建 | 模拟环境构建 |
| `复现` | 漏洞或技术复现记录 | 某漏洞复现过程 |
| `红队` | 红队攻击技术 | 横向移动、权限维持 |
| `蓝队` | 蓝队防守技术 | 日志分析、威胁狩猎 |
